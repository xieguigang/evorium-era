================================================================================
  EvoriumEra 微生物群落自然演化模拟 — 计算规则文档 v2.0
================================================================================

本文档定义了模拟程序中所有实体对象及其交互规则。
基于 v1.0 规则框架，v2.0 重点强化了代谢链完整性、交叉喂养网络、
生态位分化和群落动态，以实现更精彩的演化结果观察。

================================================================================
一、实体对象定义
================================================================================

1. 自然环境
   - 三维空间，划分为有限数量的格子（Voxel）
   - 一个格子内只能容纳一个细胞，但可容纳任意数量的分子对象
   - 环境空间可为任意形状
   - 【v2.0新增】环境具有空间梯度：氧气浓度随深度递减（表层富氧，深层缺氧），
     营养源在特定位置以热点形式分布

2. 微生物群落
   - 由不同基因组组成的微生物个体集合
   - 可因空间分布和代谢差异被划分为不同的功能群

3. 微生物（Cell）
   - 由若干个复制子序列组成的生命个体
   - 【v2.0新增】每个细胞具有"基因组维护成本"：每迭代消耗 ATP = 基因总数 × 0.5
   - 【v2.0新增】细胞具有年龄属性，年龄越大分裂概率越低

4. 复制子（Replicon）
   - 若干个基因的集合，包括基因组序列和质粒序列

5. 基因（Gene）
   - 组成复制子的基础功能单元，不关心具体位置，重点关注功能信息
   - 设定为一个功能词条

6. 功能词条（GeneOntology）— v2.0 扩展版
   降解大分子，核酸合成，复制DNA，细胞分裂，
   需氧能量代谢合成ATP，厌氧能量代谢合成ATP，
   合成抗生素，降解抗生素，基因转录，蛋白质翻译，
   细胞鞭毛运动，物质内吞，物质分泌，
   DNA整合，核酸降解，蛋白质降解，
   酸代谢，碱代谢，次级天然产物合成，固碳途径，
   群体感应，信号分子合成，细胞壁合成，生物膜合成，
   AMINO_MIX_GLU_FAMILY酶，AMINO_MIX_ASP_FAMILY酶，AMINO_MIX_SER_GLY酶，
   glucose转化酶，pyruvate酶，acetate酶，
   【v2.0新增】乳酸脱氢酶（lactate代谢），
   【v2.0新增】铁载体合成（siderophore合成，促进铁/氮源吸收）

7. 代谢物（MoleculeType）— v2.0 扩展版
   水，氧气，二氧化碳，氢离子，氢氧根离子，碳酸根离子，
   ATP，核苷酸，DNA，碳源，氮源，
   glucose，pyruvate，acetate，
   AMINO_MIX_GLU_FAMILY氨基酸，AMINO_MIX_ASP_FAMILY氨基酸，AMINO_MIX_SER_GLY氨基酸，
   抗生素，次级天然产物，信号分子，细胞壁，生物膜，
   【v2.0新增】lactate（乳酸），
   【v2.0新增】铁载体（siderophore）

二、核心代谢网络规则（v2.0 重点改进）
================================================================================

【关键设计理念】
代谢链被设计为多步骤、多分支的网络，而非单一通路。
不同物种可以专精于代谢链的不同环节，通过分泌和吸收代谢中间产物
形成交叉喂养网络。这是群落多样性和协作演化的核心驱动力。

--- 2.1 碳源代谢主链 ---

碳源(CarbonSource) ──[固碳途径]──> glucose
                                    │
                    ┌───────────────┤
                    │               │
           [glucose转化酶]    [需氧能量代谢ATP]
                    │               │
                    v               v
               pyruvate         12 ATP + 6 CO₂
                    │          (消耗1 glucose + 1 O₂)
                    │
           ┌────────┼────────┐
           │        │        │
    [pyruvate酶]  [厌氧能量  [pyruvate酶]
           │     代谢ATP]    │(乳酸发酵)
           v        │        v
       acetate      │     lactate
           │        v        │
    ┌──────┤    5 ATP       [乳酸脱氢酶]
    │      │    +2 CO₂          │
    │      │    +2 H⁺           v
    │      │                  pyruvate
    │  [acetate酶]          (回收至丙酮酸池)
    │      │
    │  ┌───┼────────┐
    │  │   │        │
    │  │ [厌氧能量  [acetate酶]
    │  │  代谢ATP]   │(TCA简化)
    │  │   │         v
    │  │   v      核苷酸
    │  │ 5 ATP    +CO₂
    │  │ +2 CO₂
    │  │ +2 H⁺
    │  │
    │  [acetate酶 → 氨基酸合成]
    │      │
    │      v
    │  AMINO_MIX_ASP_FAMILY氨基酸
    │  (消耗1 acetate + 1 氮源)
    │
    [acetate酶 → 次级代谢]
        │
        v
    次级天然产物
    (消耗2 acetate + 1 AMINO_MIX_GLU_FAMILY)

--- 2.2 氨基酸合成网络 ---

glucose + 氮源 ──[AMINO_MIX_GLU_FAMILY酶]──> AMINO_MIX_GLU_FAMILY氨基酸
    (消耗1 glucose + 2 氮源 + 1 ATP)

acetate + 氮源 ──[AMINO_MIX_ASP_FAMILY酶]──> AMINO_MIX_ASP_FAMILY氨基酸
    (消耗1 acetate + 2 氮源 + 1 ATP)

pyruvate + 氮源 ──[AMINO_MIX_SER_GLY酶]──> AMINO_MIX_SER_GLY氨基酸
    (消耗1 pyruvate + 2 氮源 + 1 ATP)

--- 2.3 交叉喂养网络的关键机制 ---

【v2.0核心新增】代谢溢流分泌（Overflow Metabolism）：
当细胞内某种代谢中间产物的数量超过细胞容量上限的60%时，
细胞会自动将超出部分的50%分泌到细胞外环境中。
受影响的分子：glucose, pyruvate, acetate, lactate, 氨基酸

这意味着：
- 拥有glucose转化酶但缺乏pyruvate酶的细胞会积累并分泌pyruvate
- 拥有pyruvate酶但缺乏glucose转化酶的细胞可以从环境中吸收pyruvate
- 不同代谢能力的细胞自然形成交叉喂养关系

【v2.0核心新增】主动分泌功能：
拥有"物质分泌"基因的细胞可以主动将特定代谢物分泌到环境中。
分泌量 = 细胞内该物质数量的20%。分泌时消耗1 ATP。

三、基因表达与蛋白质系统（v2.0 重点改进）
================================================================================

--- 3.1 基因转录规则 ---

拥有"基因转录"基因的细胞，每次执行时：
1. 从自身基因组中随机选择一个基因
2. 消耗9个核苷酸 + 1 ATP + 1 水
3. 产生1个对应功能的蛋白质（Proteins字典中对应项 +1）
4. 蛋白质具有半衰期：每5个迭代，所有蛋白质数量 -1（最低为0）
   这确保细胞必须持续表达基因才能维持功能

--- 3.2 蛋白质翻译规则 ---

拥有"蛋白质翻译"基因的细胞，每次执行时：
1. 消耗1个AMINO_MIX_GLU_FAMILY + 1个AMINO_MIX_ASP_FAMILY + 1个AMINO_MIX_SER_GLY
2. 消耗1 ATP + 1 水
3. 从自身基因组中随机选择一个基因，增加1个对应蛋白质
   （与转录协同，加速蛋白质积累）

--- 3.3 基因功能执行的前提 ---

执行任何基因功能（除需氧/厌氧能量代谢外）需要：
1. 细胞内存在对应功能的蛋白质（Proteins字典中数量 > 0）
2. 消耗1 ATP + 1 水 + 产生1 CO₂
3. 执行后对应蛋白质数量 -1（蛋白质被消耗/失活）

这意味着：细胞必须先转录和翻译基因，积累足够的蛋白质储备，
才能持续执行该功能。蛋白质的半衰期和消耗机制创造了
"基因表达-蛋白质维持-功能执行"的资源分配博弈。

================================================================================
四、能量与生存规则
================================================================================

4.1 ATP系统
- ATP上限：1000
- ATP降为0时，细胞无法执行任何动作（除需氧/厌氧能量代谢外）
- 连续5个迭代ATP为0，细胞死亡

4.2 【v2.0新增】基因组维护成本
- 每个迭代开始时，每个细胞消耗 ATP = Ceiling(基因总数 × 0.5)
- 基因越多，维持成本越高
- 这创造了基因组精简的选择压力：无用基因会被淘汰
- 最小基因组（仅含必需基因约5个）维护成本约3 ATP/迭代
- 典型基因组（15-20个基因）维护成本约8-10 ATP/迭代

4.3 细胞死亡与裂解
- 细胞死亡时，执行裂解（LyseCell）：
  所有细胞内分子释放到当前格子环境中
  包括：ATP(转化为核苷酸)、氨基酸、代谢中间产物、蛋白质(降解为氨基酸)
- 【v2.0关键】死亡细胞的代谢中间产物成为其他细胞的营养来源，
  形成"细胞裂解-营养回收-再利用"的生态循环

4.4 细胞分裂
- 条件：ATP > 500 且 核苷酸 >= 基因组核苷酸长度×2 且 细胞年龄 > 10
- 消耗：核苷酸(基因组长度×2) + ATP(100) + 水(10)
- 母细胞保留60%分子，子细胞获得40%
- 子细胞基因组复制（含突变概率）
- 分裂后母细胞年龄重置为0

================================================================================
五、环境规则（v2.0 重点改进）
================================================================================

5.1 【v2.0新增】氧气梯度
- 环境中氧气浓度随Z轴（深度）递减
- 表层（Z=0）：氧气充足，每迭代自动补充至200单位/格子
- 中层（Z=Depth/3 ~ 2Depth/3）：氧气中等，每迭代补充至50单位/格子
- 深层（Z>2Depth/3）：氧气稀少，每迭代补充至5单位/格子
- 这创造了天然的好氧/厌氧生态位分化

5.2 【v2.0新增】营养源热点与补充
- 碳源：在环境底部和侧壁特定位置设置热点，每迭代补充至100单位
- 氮源：在环境随机位置设置3-5个热点，每迭代补充至50单位
- 葡萄糖：在碳源热点附近，每迭代补充至30单位
- 水：全环境每迭代补充至50单位/格子

5.3 扩散规则（v2.0 扩展）
- 参与扩散的分子类型扩展为：
  水、氧气、二氧化碳、氢离子、氢氧根离子、碳酸根离子、
  碳源、氮源、glucose、pyruvate、acetate、lactate、
  信号分子、抗生素、铁载体
- 扩散量 = 浓度差 × 0.15（从0.1提高至0.15，加速物质交换）
- 每次扩散量上限为5个单位
- 生物膜覆盖的格子：扩散速率降低至0.03（几乎隔绝）

5.4 被动吸收
- 小分子（水、氧气、CO₂、H⁺、OH⁻、碳源、氮源）可自由穿透细胞膜
- 每迭代自动从环境进入细胞，数量 = min(环境浓度, 细胞容量剩余)
- 进入量上限：每类分子每迭代最多20单位

================================================================================
六、运动与水平基因转移
================================================================================

6.1 鞭毛运动
- 拥有"细胞鞭毛运动"基因且有对应蛋白质时
- 随机移动到相邻空格子
- 消耗2 ATP + 1 水
- 【v2.0新增】趋化性：优先向营养浓度高的方向移动（概率加权）

6.2 水平基因转移（HGT）
- 相邻两个细胞都有"DNA整合"基因时
- 有5%概率交换一个质粒
- 【v2.0新增】无质粒的细胞可以从死亡细胞残留的DNA中获取基因片段
  （需要"DNA整合"基因，概率2%）

================================================================================
七、突变规则（v2.0 改进）
================================================================================

7.1 突变率
- DNA复制时突变率：3%（从1%提高，加速演化）
- 基因组越小突变率越低（精简基因组的优势）

7.2 突变类型
- 缺失突变（30%概率）：随机删除一个基因
- 重复突变（30%概率）：随机复制一个已有基因
- 功能突变（30%概率）：随机改变一个基因的功能
- 【v2.0新增】质粒获得（10%概率）：随机获得一个含1-3个基因的质粒

7.3 【v2.0新增】突变偏向性
- 功能突变时，新功能从以下加权分布中选取：
  代谢类基因（glucose/pyruvate/acetate酶等）：权重3
  氨基酸合成类基因：权重2
  看家基因（转录/翻译/复制）：权重1
  其他基因：权重1
- 这确保突变更可能产生代谢多样性而非无用基因

================================================================================
八、群体感应与生物膜
================================================================================

8.1 信号分子合成
- 拥有"信号分子合成"基因时
- 消耗1碳源 + 1 AMINO_MIX_SER_GLY → 1信号分子
- 信号分子自动分泌到环境中（扩散性分子）

8.2 群体感应
- 拥有"群体感应"蛋白质且信号分子>50时
- Hill函数激活：P = S²/(K²+S²)，K=80
- 激活后：次级天然产物合成或生物膜合成必定执行一次

8.3 生物膜
- 合成：10氮源 + 5 AMINO_MIX_ASP_FAMILY → 1生物膜
- 生物膜覆盖的格子扩散速率大幅降低
- "降解大分子"蛋白质可降解生物膜
- 【v2.0新增】生物膜内的细胞获得抗生素抗性加成（抗生素伤害减半）

================================================================================
九、抗生素与竞争
================================================================================

9.1 抗生素合成
- ATP > 800 时，"合成抗生素"基因表达概率提高
- 消耗2 acetate + 1 AMINO_MIX_GLU_FAMILY → 1抗生素
- 抗生素分泌到环境中

9.2 抗生素作用
- 环境中有抗生素时，无"降解抗生素"蛋白质的细胞每迭代损失3 ATP
- 有"降解抗生素"蛋白质的细胞每迭代仅损失1 ATP
- 【v2.0新增】生物膜内细胞抗生素伤害减半

9.3 降解抗生素
- 拥有"降解抗生素"蛋白质时
- 消耗1 ATP，移除当前格子1单位抗生素

================================================================================
十、酸碱环境响应
================================================================================

10.1 酸代谢
- H⁺ >= 10 时触发
- 消耗1 H⁺ + 1 碳源 → 1 水 + 1 ATP

10.2 碱代谢
- OH⁻ >= 10 时触发
- 消耗1 OH⁻ + 1 氮源 → 1 水 + 1 ATP

10.3 【v2.0新增】pH生态位
- 厌氧代谢产生H⁺，使局部环境酸化
- 酸化环境抑制无酸代谢能力的细胞
- 这创造了酸耐受型和非酸耐受型的生态位分化

十一、初始化规则（v2.0 改进）
================================================================================

11.1 环境初始化
- 100×100×100 空间（可配置）
- 氧气按深度梯度填充
- 碳源/氮源/葡萄糖在热点位置填充
- 水全环境填充至50单位/格子

11.2 细胞初始化
- 初始细胞数量：100（从50增加）
- 每个细胞随机基因组：8-15个基因
- 【v2.0关键】初始基因组保证包含至少：
  1个能量代谢基因（随机选择好氧或厌氧）
  1个基因转录基因
  1个蛋白质翻译基因
  1个物质内吞或物质分泌基因
  其余基因随机分配（偏向代谢类基因）
- 初始ATP：200
- 初始分子：随机分配基础代谢物

11.3 【v2.0新增】初始多样性保证
- 初始细胞被分为3-5个"创始种群"
- 每个种群共享一组核心基因但有不同的代谢基因组合
- 例如：种群A专精glucose→pyruvate，种群B专精pyruvate→acetate，
  种群C专精厌氧acetate代谢
- 这确保初始就存在交叉喂养的潜力

================================================================================
十二、迭代计算流程
================================================================================

每一步迭代的执行顺序：

1. 【环境更新】补充氧气（按深度梯度）、营养源（按热点位置）、水
2. 【扩散】所有格子间分子扩散
3. 【被动吸收】小分子自动进入细胞
4. 【基因组维护】每个活细胞扣除基因组维护ATP成本
5. 【蛋白质衰减】所有蛋白质半衰期递减
6. 【随机打乱】打乱所有格子的处理顺序
7. 【逐格处理】对每个含活细胞的格子：
   a. 基因表达（转录+翻译），最多执行MaxCellActions次
   b. 按优先级执行基因功能：
      - ATP < 50%上限时，能量代谢基因优先
      - 氧气 < 10时，厌氧代谢优先
      - 有抗生素时，降解抗生素优先
      - 信号分子 > 阈值时，群体感应激活
      - 其余功能按蛋白质丰度加权随机选择
   c. 代谢溢流检查：超出60%容量的中间产物自动分泌50%
   d. 物质运输（内吞/分泌）
   e. 运动（鞭毛）
   f. 细胞分裂检查
8. 【死亡检查】ATP连续为0达5次的细胞死亡并裂解
9. 【HGT检查】相邻细胞间质粒交换
10.【突变】分裂的细胞进行基因突变
11.【快照记录】按配置间隔记录快照

================================================================================
十三、预期可观察的演化现象
================================================================================

通过上述规则改进，预期可以观察到：

1. 代谢分工与交叉喂养
   - 葡萄糖专化型：拥有glucose转化酶，将glucose→pyruvate，分泌多余pyruvate
   - 丙酮酸利用型：吸收环境pyruvate，转化为acetate或lactate
   - 乙酸专化型：利用acetate进行厌氧产能或氨基酸合成
   - 这三种类型形成经典的交叉喂养链

2. 好氧-厌氧分层
   - 表层：好氧菌主导，高效产能，快速繁殖
   - 深层：厌氧菌主导，利用上层分泌的acetate和lactate

3. 基因组精简与扩展的博弈
   - 资源丰富时：基因组扩展，获得更多功能
   - 资源匮乏时：基因组精简，降低维护成本
   - 专化型细胞基因组较小，通用型细胞基因组较大

4. 群体感应与生物膜形成
   - 高密度区域触发群体感应
   - 生物膜形成创造局部微环境
   - 膜内细胞协作抵抗抗生素

5. 抗生素军备竞赛
   - 资源竞争激烈时，部分细胞开始合成抗生素
   - 其他细胞演化出抗性或逃避行为
   - 形成生产者-抗性者的动态平衡

6. 营养循环
   - 死亡细胞裂解释放营养
   - 降解大分子的分解者角色出现
   - 形成生产者-消费者-分解者的生态循环
